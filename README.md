# Ljudlådan

Sound sensitivity tool with custom sound profiles

Part of [Autismappar](https://www.autismappar.se) — free apps for autism, language disorders and ADHD.

## Install

### Ubuntu/Debian
```bash
sudo apt install ljudladan
```

### Fedora
```bash
sudo dnf install ljudladan
```
